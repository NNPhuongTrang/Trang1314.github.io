How to implement a Smart traffic light using STM32

  • Qui Nguyen

The project involves using the STM32F103RB to simulate a traffic light system at an intersection with a pedestrian button. The system has three modes and a total of four buttons.
The four buttons operate as follows:

  • Button 0 - Pedestrian button: This button operates in both Auto mode and Manual mode. When pressed, the pedestrian light will display the opposite signal of the traffic light on the road. The buzzer will not sound in Manual mode. In Auto mode, the buzzer will sound when the pedestrian light is green, and the sound will increase in volume as the time left for the green light decreases. The buzzer will automatically turn off at the end of the second yellow light cycle.
  • Button 1 - Mode button: Press this button to cycle through the modes in the following order: Auto mode -> Manual mode -> Tuning mode.
  • Button 2 - Switching button: This button switches the state of the traffic light between Manual mode and Tuning mode.
  • Button 3 - Tuning button: In Tuning mode, press this button to adjust the time that each traffic light is on.

The three modes operate as follows:

  • Auto mode: The traffic lights operate according to a normal cycle. Press Button 1 to switch to Manual mode.
  • Manual mode:
    When switching from Auto mode to Manual mode, the traffic light state remains the same as in Auto mode.
    When Button 2 is pressed, the system switches to the next traffic light state in the cycle from Auto mode.
    If Button 2 is not pressed within 30 seconds, the system will return to Auto mode, and the traffic light state will be the next state in the cycle.
    Pressing Button 1 will switch to Tuning mode.
  • Tuning mode:
    Tuning mode is used to adjust the time values. Note: Red1 = Green2 + Yellow2, Red2 = Green1 + Yellow1 (Red >= 2)
    The values to be adjusted in sequence are Red1, Red2, Green1, Green2, Yellow1, Yellow2. Red1 will be adjusted from [2 -> 99]. If Red1 <= Yellow2, then Yellow2 will be reset to 1, and Green2 will be set to Red1 - Yellow2. The same is true for Red2.
    Green1 will be adjusted from [1 -> (Red2 - 1)], and Yellow1 will be set to Red2 - Green1. The same is true for Green2.
    Yellow1 will be adjusted from [1 -> (Red2 - 1)], and Green1 will be set to Red2 - Yellow1. The same is true for Yellow2.
    Pressing Button 1 will switch back to Auto mode.
